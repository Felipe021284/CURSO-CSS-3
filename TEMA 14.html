INICIO

La propiedad background-size

La propiedad background-size añade nueva funcionalidad a CSS que nos permite especificar el tamaño de imágenes de fondo, utilizando ya sea longitudes o porcentajes.

ejemplo de background-size

NOTA

Las versiones actuales de los navegadores más populares - incluyendo Firefox, Safary, Chrome, Internet Explorer y Opera - ahora soportan background-size, sin la necesidad de prefijos de proveedor.

Los valores para background-size

Los otros dos valores posibles para el tamaño del fondo son las palabras clave: "contain" (contiene) y "cover" (cubre).

La palabra clave "contain" escala la imagen de manera que se ajuste al contenedor.
En otras palabras, la imagen crecerá o se encogerá proporcionalmente, pero el ancho y la altura no excederán las dimensiones del contenedor:

ejemplo de background-size contain

background-size también acepta la palabra clave "cover". La imagen es escalada para ajustarse a todo el contenedor, sin embargo, si tiene una relación de aspecto diferente, la imagen será recortada:

ejemplo de background-size cover
