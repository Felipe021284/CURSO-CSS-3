La propiedad de CSS opacity provee un excelente medio para añadir opacidad a cualquier elemento.

nota

El valor de la propiedad "opacity" debe ser un número entre 0.0 (completamente transparente) y 1.0 (completamente opaco).

Opacidad en Internet Explorer

Para que la propiedad "opacity" funcione en todas las versiones de IE, utiliza filter:alpha(opacity=x) junto con la propiedad "opacity". x puede ser un valor entre 0 y 100.
El valor 0 resulta en un elemento completamente transparente (por ejemplo, 100% transparente), mientras que el valor 100 hace que el elemento sea completamente opaco (por ejemplo, 0% transparente).
