INTRODUCCION A CSS3

CSS3 es el último estándar de CSS. CSS3 es totalmente compatible con versiones anteriores de CSS.

Algunas de las novedades más importantes son:
Border radius
nos permite crear esquinas redondeadas para los elementos.
Border images
nos permite especificar una imagen como el borde alrededor de un elemento.
Multiple backgrounds
aplica fondos múltiples a los elementos.
Animaciones y efectos
mucho más!

CSS3: Nuevas características

Para hacer más fácil y rápido el desarrollo web, CSS3 introduce nuevas características adicionales, incluyendo las siguientes:

Sombra de caja
Con la propiedad box-shadow, puedes añadir una o más sombras a un elemento especificando valores para el color, tamaño, desenfoque y desplazamiento.

Degradados
Los gradientes o degradados en CSS3 nos permiten establecer el color de fondo del elemento a un degradado. Hay dos tipos de degradados disponibles: Lineal y Radial.

Las transformaciones te permiten rotar, escalar, mover y sesgar elementos.

Otra característica popular son las Transiciones que te permiten animar desde un valor de propiedad CSS a otro. Puedes combinarla con transformaciones y animar la posición, rotación o escala del elemento.

Las propiedades que atraen la mayor atención son las animaciones.
Las animaciones en CSS tienen sus propias especificaciones y te permiten crear fotogramas clave, establecer la duración, suavizado y más.

NOTA

Muchas de las nuevas características han sido implementadas por los principales navegadores, por lo que ya puedes disfrutar el poder de CSS3.
