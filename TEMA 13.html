Degradados radiales

Para crear un degradado radial, se debe definir al menos dos paradas de color.
El degradado radial se define por su centro.

El primer valor define la posición del degradado. Podemos utilizar una palabra clave descriptiva, como top (arriba), bottom (abajo), center (centro) o izquierda (left); o podemos especificar, por ejemplo, 50% 50% para establecer el gradiente en el centro o 0% 0% para definir el degradado desde la parte superior izquierda.

El segundo valor define la forma y el tamaño del degradado. Hay dos argumentos para darle forma al degradado: El primero es la ellipse, que es por defecto; y el segundo es circle (círculo).

Por último, el tercer valor define la combinación de colores.

Fijando las formas

El parámetro "forma" define la forma. Si no defines la forma del degradado radial, tomará el valor ellipse por defecto.

Posición del degradado radial

Esencialmente, podemos utilizar el mismo método que usamos para especificar la ubicación de una imagen de fondo con la propiedad background-position de CSS para especificar la ubicación del centro de la elipse. Primero especificamos la posición horizontal del fondo, y -opcionalmente- la posición vertical utilizando alguna de las palabras claves (left, center right, o top, center, bottom), valores de longitud, valores porcentuales, o alguna combinación de estos.

Fijando las paradas para los colores
