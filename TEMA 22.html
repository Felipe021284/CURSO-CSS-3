El método scale()

El < método scale() aumenta o reduce el tamaño de un elemento, de acuerdo a los parámetros dados para el ancho y el alto.
1 equivale al tamaño original,
2 para el doble del tamaño original,
y así sucesivamente.

En el siguiente ejemplo, hemos disminuido el primer div por el factor 0,7 tanto horizontal como verticalmente y aumentamos el segundo div por un factor de 1.5 horizontal y verticalmente.

scale 0.7

scale 1.5
nota

Si sólo se pasa un parámetro al método scale(), se aplicará ese factor tanto para la altura como para la anchura.
